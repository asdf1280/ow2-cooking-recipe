Supported colors

## How to contribute

If there's a missing color, please add it to the list below, and add appropriate css class named .item-color-<CODE> to index.html > <style> section.

If a menu/item shown in the recipe has neither background nor text color, it means the color's missing. Contributions are welcome.

There are several colors missing that were added on the last update. They should be added.
